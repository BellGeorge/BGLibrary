CouchChangeTracker Class Reference

Inherits from NSObject
Conforms to NSStreamDelegate
Declared in CouchChangeTracker.h

Overview

Reads the continuousmode _changes feed of a database, and sends the individual change entries to its client’s -changeTrackerReceivedChange:.

This class is used internally by CouchDatabase and you shouldn't need to use it yourself.
