image

Observe this property to be notified when the image has finished loading.

@property (nonatomic, strong) UIImage *image

Discussion

Observe this property to be notified when the image has finished loading.

If initWithURL:placeholderImage initializer was used, this will return the placeholder image until the image at url has finished loading.

If there’s an error loading and the initWithURL:placeholderImage initializer was used, this will continue to return the placeholder image even after the error occurs.
